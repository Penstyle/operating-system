1. Overview
This assignment implements:

TCP client and server

UDP client and server

A packet sniffer that detects and parses IPv4 TCP/UDP packets

All programs communicate over the loopback interface (127.0.0.1) and use ports:

8080 for TCP

8081 for UDP

The sniffer listens on the loopback interface and prints packet details when traffic is detected.

4. How to Run the Programs
4.1 TCP Communication
Terminal 1 – Start TCP Server
bash
python3 tcp_server.py
Terminal 2 – Run TCP Client
bash
python3 tcp_client.py
Expected behavior:

Client sends "TCP_TEST_PACKET"

Server prints the received message

Client prints acknowledgment from server

4.2 UDP Communication
Terminal 1 – Start UDP Server
bash
python3 udp_server.py
Terminal 2 – Run UDP Client
bash
python3 udp_client.py
Expected behavior:

UDP server prints sender address and "UDP_TEST_DATAGRAM"

5. Running the Packet Sniffer
The sniffer must run with elevated privileges.

Terminal – Start Sniffer
bash
sudo python3 sniffer.py
The sniffer listens on:

lo (loopback interface)

Ports 8080 (TCP) and 8081 (UDP)

When traffic is generated by the clients, the sniffer prints:

Source IP

Destination IP

Source Port

Destination Port

Protocol type (TCP or UDP)

6. Testing the Sniffer
